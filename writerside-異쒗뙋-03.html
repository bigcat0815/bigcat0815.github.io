3. Writerside 프로젝트용 배포판 설정

  1. 배포하기 위한 저장 bin 폴더를 생성

  2. menu | run | edit configurations | web achieve

  3. 위 설정에 가서 출력 위치를 bin으로 설정한다.

  4. 실행 시키면 배포판이 압축되어 bin에 저장된다.

  5. git commit / push를 하면 압축파일까지 writerableProject에 포함되서 올라간다.
