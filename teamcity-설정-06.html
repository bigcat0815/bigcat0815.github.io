TeamCity 설정 1단계 06

Teamcity는 베타서비스인 cloud 무료 버전이다.

6. TeamCity Job 설정 1

  1. 계정 settings | Developer settings | Personal access tokens | Tokens (classic) 이동

  2. Create Pipeline을 선택하면 writerside project 계정을 선택한다.

  3. Job 1 선택

  4. Steps -> SCRIPT 선택

  5. Step name: Extract ZIP 의미있는 이름으로 지정하자

  6. Working directory: bin/ 배포압축파일이 bin 폴더에 있다.

    - 폴더 모양을 눌러서 폴더 위치를 지정할 수있으니 참고 하자

  7. script content: unzip -O CP949 *.zip -d result

    -O CP949 는 한글 파일명칭을 인코딩하기 위함

    *.zip -d result 는 현재 위치의 모든 zip을 result 폴더에 압축 해제
